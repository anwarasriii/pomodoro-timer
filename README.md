
# 🕒 Pomodoro Timer App

A simple and clean Pomodoro Timer built with **React.js** to help improve focus and productivity using the Pomodoro technique.

---

## 🚀 Features

* ⏱️ 25-minute work timer
* ☕ 5-minute break timer
* ▶️ Start / Pause / Reset controls
* 🔄 Auto switch between Work and Break sessions
* 📊 Progress bar showing session completion
* 🎯 Clean and minimal UI

---

## 🛠️ Built With

* React.js (Hooks: `useState`, `useEffect`)
* JavaScript (ES6+)
* CSS (custom styling)

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/anwarasriii/pomodoro-timer.git
```

Navigate to project folder:

```bash
cd pomodoro-timer
```

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm run dev
```
